# Pulseboard

Self-hosted мониторинг доступности со статус-страницами: мониторы, автоматические
инциденты, алерты (email/Telegram), real-time-дашборд и публичные статус-страницы.

Стек: Laravel 13, Vue 3 (Composition API, TypeScript), Vite, Tailwind CSS, Sail
(MySQL, Redis, Mailpit), Reverb (WebSockets).

Проект в активной разработке — полное описание с архитектурой, ER-диаграммой и
скриншотами появится по завершении.
